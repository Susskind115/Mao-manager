# Time Manager (Vite + React)

一个纯前端（本地版）的时间管理/任务积分网站：
- ✅ 任务系统：勾选任务获得积分（每日上限 150）
- 🛒 商店系统：用积分兑换自定义奖励
- 📜 记录页：按天统计 & 兑换记录
- 💾 备份读档：导出/导入 JSON（防止清理浏览器数据导致丢失）

> 数据默认保存在浏览器 LocalStorage（单机本地版）。换设备不会同步。

---

## 1) Windows 本地运行

### 前置条件
安装 Node.js（建议 LTS）

### 运行
在项目根目录（看到 package.json 的那一层）打开终端：

```bash
npm install
npm run dev
```

---

## 2) 生成可部署版本（dist）

```bash
npm run build
```

---

## 3) 部署（最省事：Netlify 拖拽 dist）
1. `npm run build`
2. 把 `dist/` 拖到 Netlify 的手动部署页面即可

> 本项目使用 HashRouter（URL 形如 `/#/tasks`），因此静态托管一般无需额外重定向配置。

---

## 4) 数据备份/读档（推荐）
在“记录”页：
- **导出备份 JSON**：下载备份文件（建议保存到网盘/USB）
- **导入并恢复**：选择备份文件导入（会覆盖当前本地数据）


## 5) 新功能：悬赏任务
- 一次性任务，积分无上限
- 有截止日期（deadline）
- 完成后自动下架
- 计入历史总获得（不受每日 150 封顶影响）

## 6) 新功能：下架后可删除
- 任务/商店奖励在“已停用/已下架”列表里支持删除（软删除，不影响历史统计）

## 7) 新功能：等级系统
- 基于历史总获得积分
- 支持自定义每个大等级名称、跨度、段数（小等级数）


## 页面结构（建议）
- ✅ 日常任务：`/#/tasks`（可重复打卡，受每日 150 上限）
- 🏹 悬赏任务：`/#/bounties`（一次性，积分无上限，有截止日期，完成后自动下架）
